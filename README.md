# School Management System (C#, .NET, MySQL)

A **desktop application** for efficiently managing school operations, including students, teachers, and fee records. Built using C# and .NET with MySQL as the database backend.  

## Features

- **Student Management**: Add, edit, delete, and view student records.
- **Teacher Management**: Maintain teacher profiles and assignments.
- **Fee Management**: Track student fee payments and generate reports.
- **Search & Filter**: Quickly search for students or teachers.
- **User-Friendly Interface**: Built with a responsive desktop UI using Windows Forms / WPF.
- **Persistent Storage**: All data stored in a MySQL database.

## Tech Stack

- **Frontend & Backend**: C# with .NET Framework / .NET Core
- **Database**: MySQL
- **IDE**: Visual Studio (recommended)
- **UI**: Windows Forms or WPF
